🧠 나만의 퀴즈 게임 (Python)
📌 프로젝트 개요
본 프로젝트는 Python을 활용하여 CLI 기반의 퀴즈 게임을 구현하는 것을 목표로 한다.
사용자는 퀴즈를 풀고, 새로운 문제를 추가하며, 점수를 확인할 수 있다.
또한 JSON 파일을 통해 데이터를 저장하고 불러오는 기능을 구현하였다.

⚙️ 주요 기능
1. 퀴즈 풀기
* 저장된 퀴즈를 순서대로 출력
* 사용자 입력을 통해 정답 확인
* 점수 계산 및 출력
2. 퀴즈 추가
* 문제 입력
* 선택지 4개 입력
* 정답 번호 입력 (1~4)
* 새로운 퀴즈 리스트에 추가
3. 퀴즈 목록 보기
* 현재 저장된 퀴즈 목록 출력
4. 점수 확인
* 최고 점수 출력
* 최고 점수 갱신 기능 포함
5. 데이터 저장 및 불러오기
* JSON 파일(state.json)을 이용한 데이터 저장
* 프로그램 실행 시 자동 로드
* 종료 시 자동 저장

🧩 클래스 구조
Quiz 클래스
* 문제(question), 선택지(choices), 정답(answer) 저장
* 문제 출력(show)
* 정답 확인(check_answer)
QuizGame 클래스
* 전체 게임 흐름 제어
* 메뉴 출력 및 사용자 입력 처리
* 퀴즈 실행, 추가, 조회, 점수 관리

💾 데이터 구조 (JSON)
{
    "quizzes": [
        {
            "question": "문제",
            "choices": ["보기1", "보기2", "보기3", "보기4"],
            "answer": 1
        }
    ],
    "best_score": 2
}

▶️ 실행 방법
python game.py

📚 사용 기술
* Python
* JSON (파일 저장 및 불러오기)

🔧 개선 사항 (향후 계획)
* 문제 삭제 기능 추가
* 문제 수정 기능 추가
* GUI 기반 인터페이스 구현

🙌 느낀 점
클래스와 객체를 활용하여 데이터를 구조적으로 관리하는 방법을 학습할 수 있었다.
또한 JSON을 활용한 데이터 저장 및 불러오기 기능을 구현하며 파일 입출력에 대한 이해를 높일 수 있었다.
